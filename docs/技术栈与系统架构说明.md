# 校园影院购票与排片管理系统

## 一、项目技术栈

本项目采用前后端分离架构开发，实现校园影院电影浏览、在线选座购票、后台电影管理、影厅管理、排片管理以及订单管理等功能。

### 1. 前端技术栈

前端采用 Vue3 + Vite 开发。

主要技术包括：

- Vue 3
- Vite
- Vue Router
- Axios
- Element Plus
- CSS3

前端负责：

- 首页电影展示
- 电影详情展示
- 在线选座
- 后台管理页面
- 数据交互与展示

项目目录：

frontend/campus-cinema-web

---

### 2. 后端技术栈

后端采用 ASP.NET Core Web API 开发。

主要技术包括：

- ASP.NET Core 9.0
- Entity Framework Core
- RESTful API
- Swagger
- CORS跨域配置

主要功能：

- 电影管理接口
- 影厅管理接口
- 排片管理接口
- 订单管理接口
- 数据库访问与业务逻辑处理

项目目录：

backend/CampusCinema.Api

---

### 3. 数据库技术栈

数据库采用 Microsoft SQL Server。

开发过程中使用 Docker 部署 SQL Server 容器。

主要数据表：

- Movies（电影表）
- Halls（影厅表）
- Schedules（排片表）
- Orders（订单表）

数据库通过 Entity Framework Core Migration 管理。

---

## 二、系统架构

系统采用前后端分离架构。

用户浏览器

↓

Vue3 前端

↓

Axios HTTP请求

↓

ASP.NET Core Web API

↓

Entity Framework Core

↓

## 五、项目运行方式

启动数据库：

```bash
docker start campus-sql
```

启动后端：

```bash
cd backend/CampusCinema.Api
dotnet run
```

启动前端：

```bash
cd frontend/campus-cinema-web
npm run dev -- --host 0.0.0.0
```

本项目采用 Vue3 + ASP.NET Core Web API + SQL Server 技术栈实现校园影院购票与排片管理系统。通过前后端分离架构完成电影管理、影厅管理、排片管理、订单管理等核心业务功能，具有较好的扩展性和维护性。
